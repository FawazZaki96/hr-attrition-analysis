# 📁 Data Description

## 📊 Dataset: HR_Analytics.csv

This dataset contains employee-level information used to analyze attrition patterns.

---

## 🔑 Key Columns

* **Age** → Employee age
* **Attrition** → Whether employee left (Yes/No)
* **Department** → Department of employee
* **JobRole** → Specific role
* **MonthlyIncome** → Salary level
* **JobSatisfaction** → Satisfaction score (1–4)
* **WorkLifeBalance** → Work-life balance score (1–4)
* **OverTime** → Overtime status (Yes/No)
* **YearsAtCompany** → Employee tenure

---

## 🧠 Notes

* No major data quality issues found
* "StandardHours" column was constant and not useful
* Dataset was prepared for analysis in Excel before visualization

---

## 📌 Purpose

The dataset is used to:

* Identify high-risk employee segments
* Analyze attrition drivers
* Support business recommendations
