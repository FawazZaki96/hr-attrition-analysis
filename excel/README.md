# 📊 Excel Analysis

## 🔧 Purpose

Excel was used for:

* Data cleaning
* Feature engineering
* Initial analysis

---

## 🧹 Data Preparation

* Removed duplicates (none found)
* Checked for missing values
* Created new columns:

  * **Income_Group**
  * **Age_Group**
  * **Tenure_Group**

---

## 📊 Analysis Performed

* Attrition rate calculation
* Segmentation analysis:

  * Age vs Attrition
  * Income vs Attrition
  * Tenure vs Attrition
* Pivot tables for:

  * Attrition distribution
  * Segment-level insights

---

## 🧠 Key Findings from Excel

* Attrition rate: ~16%
* Early-tenure employees (<2 years) show highest attrition
* Low-income employees are more likely to leave

---

## 📌 Output

Cleaned dataset and structured insights were used to build the Power BI dashboard.
